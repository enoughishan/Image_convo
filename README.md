# Face Sketch Plotter (Beginner Version)

## Overview

Face Sketch Plotter is a simple Python-based project that converts a face image into different sketch-style visualizations.  
The project is written in a beginner-friendly manner and avoids advanced programming concepts so that it is easy to understand, explain, and modify.

This project is suitable for:
- College minor projects
- Lab submissions
- Practical exams
- Viva and project demonstrations

---

## Features

- Upload any face image
- Convert image into multiple sketch styles
- Generate visual plots from sketch data
- View image processing steps clearly
- Simple web interface using Gradio

---

## Sketch Styles Available

- **Basic** – Simple edge detection
- **Detailed** – Shaded sketch with edges
- **Cartoon** – Black and white comic-style sketch
- **Pencil Sketch** – Hand-drawn pencil effect
- **Stipple** – Dot-based sketch style

---

## Technologies Used

- Python  
- OpenCV (image processing)
- NumPy (array operations)
- Matplotlib (plotting and visualization)
- Gradio (web interface)
- Pillow (image handling)
